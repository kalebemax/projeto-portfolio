Meus Projetos

Previsão do tempo

Uma página de previsão do tempo para cidades. Essa página se conecta com uma API que fornece informações meteorológicas do dia atual e dos cinco dias futuros. Tecnologias utilizadas: React.js, JavaScript, HTML e CSS.

Clone da Netflix

O site é um clone da interface da Netflix. Ele apresenta um design moderno, com um layout responsivo e seções para exibir filmes e séries. Tecnologias utilizadas: JavaScript, HTML, CSS e Bootstrap.

Site de Avaliação

Os usuários podem selecionar uma nota e enviá-la. Após o envio, o sistema exibe uma mensagem de agradecimento junto com a nota escolhida. O design é responsivo para melhorar a experiência do usuário. Tecnologias utilizadas: JavaScript, HTML e Tailwind CSS.

Coming Soon

O site é uma página de "Coming Soon" para a marca fictícia Base Apparel. Ele possui um design moderno e responsivo, com um campo de inscrição por email. Tecnologias utilizadas: JavaScript, HTML e Tailwind CSS.
